CLI Command Reference

throughput_central

Throughput Central
throughput_central stop cli_throughput_central_stop
Stops remote transmission
Alias central x
throughput_central start u8 cli_throughput_central_start
Starts remote transmission
Alias central s
Arguments
  • u8Type: 1: notification, 2: indication
throughput_central status cli_throughput_central_status
Prints current status
Alias central t

throughput_central central_mode

Mode
throughput_central central_mode set u8 u32 cli_throughput_central_mode_set
Set reception mode
Alias central m s
Arguments
  • u8Mode: 0:Continous, 1:Fixed time, 2: Fixed size
  • u32Value to set (in case of fixed time/size)
throughput_central central_mode get cli_throughput_central_mode_get
Get reception mode, 0:Continous, 1:Fixed time, 2: Fixed size
Alias central m g

phy

PHY settings
phy scan_set u8 cli_throughput_central_phy_scan_set
Set PHY used for scanning
Alias y s
Arguments
  • u8Scan PHY, 1:1M 4:Coded PHY
phy conn_set u8 cli_throughput_central_phy_conn_set
Set PHY used for the connection, 1:1M 2:2M 4:Coded 125k, 8:Coded 500k PHY
Alias y c
Arguments
  • u8Connection PHY
phy get cli_throughput_central_phy_get
Read PHY settings for scan and connection, 1:1M 2:2M 4:Coded 125k, 8:Coded 500k PHY
Alias y g

connection

Connection settings
connection set u32 u32 u32 u32 cli_throughput_central_connection_set
Set connection settings
Alias c s
Arguments
  • u32Minimum connection interval (in 1.25 ms steps)
  • u32Maximum connection interval (in 1.25 ms steps)
  • u32Responder latency (in connection intervals)
  • u32Supervision timeout (in 10 ms steps)
connection get cli_throughput_central_connection_get
Read connection settings
Alias c g
